展示&交互说明标注

G全局触发与消失逻辑
触发条件：未登录时以设备设备为维度，登录后以账号维度，未完成任一路径的账号，每次启动时展示
消失条件：永久1次
覆盖范围： 匿名&旧版登录设备均覆盖
登录页改造（游客模式）
21"游客模式，先体验一下"入口
位置：登录弹窗底部，三方登录按钮下方

触发条件（显示规则）：
• 仅在未登录过的设备显示
• 已登录过的设备不显示此入口
• 新增改造点：现有登录页无此入口

登录页输入规则：
• 手机号输入框最多支持11位数字
• 验证码6位数字

点击行为：
• 自动创建游客账号，跳转到【欢迎首屏】引导页

游客账号规则：
• 有效期：永久
• 数据：继承保留，不丢失
• 清理策略：超过90天未登录，可清理掉数据
• 允许参与运营活动和签到

转正场景（触发拉起正式登录）：
• Steam 账号绑定
• Epic 账号绑定
• GOG 账号绑定
• 云游戏充值
• 云游组队玩（创建/加入房间）

主动转正：游客可随时通过个人中心"前往登录"绑定手机号升级为正式账号 可交互新增
页面1：欢迎首屏
1Logo + 标题区
纯展示，不可交互。标题"欢迎来到盖世游戏"，副标题"Mac上畅玩Windows游戏"
2"我有游戏想玩"选项
点击 → 跳转到【游玩方式选择页】
目标用户：已有Steam账号或本地游戏文件（含CrossOver用户）
描述文案："已有Steam账号或游戏文件，想在Mac上玩"可交互
3"我是新手"选项
点击 → 跳转到【新手秒玩列表页】
目标用户：纯新用户，从未玩过
描述文案："没玩过，想体验一下热门游戏"可交互
4"跳过"按钮
点击 → 关闭引导弹窗，直接进入探索页
状态记录：跳过不记录为已完成，下次登录仍触发引导注意
5平台特色介绍区
纯展示，运营可配置
国内版（3条）：
•与 Steam、Epic、GOG平台数据互通
• 支持云游戏，无需下载一键秒玩
• 全游戏适配AI超级插帧技术
海外版（2条）：隐藏"云游戏"条目国内/海外差异
页面2：游玩方式选择
6返回按钮
点击 → 返回到【欢迎首屏】
7"导入游戏"卡片
点击 → 进入【自动扫描页】
扫描范围：
1. ~/Library/.../CrossOver/Bottles/Steam/
2. ~/Library/.../Steam/steamapps/
3. /Volumes/（外接硬盘优先）
流程：自动扫描，无需用户手动选择路径可交互
8"绑定Steam"卡片
点击 → 拉起Steam登录流程
前置条件：需先登录平台账号（游客不可用）
成功后：进入探索页，引导标记为已完成
失败处理：提示"绑定失败，请重试"可交互
9"跳过"按钮
同标注4逻辑。点击关闭弹窗进入探索页，不记录为已完成
页面3：扫描中
10扫描动画 + 进度
行为：自动执行，无需用户操作
扫描路径顺序：
1. CrossOver容器（优先）
2. Steam steamapps目录
3. 外接硬盘 /Volumes/
超时处理：10秒未完成 → 提示"扫描超时，请手动选择"
预计耗时：3-15秒自动流程
页面4：扫描结果
11游戏列表（可勾选）
每条显示：勾选框 + 游戏封面 + 游戏名 + 来源路径 + 文件大小
默认状态：全部选中
交互：点击单条可取消/重新选择
来源标识：CrossOver Steam / steamapps（区分展示）可交互
12全选/取消全选
逻辑：当前全选 → 点击变为"取消全选"；当前非全选 → 点击变为"全选"
联动：更新底部已选计数可交互
13"导入并进入游戏库"按钮
点击 → 导入选中游戏 → 完成后进入探索页游戏库tab
引导状态：标记为已完成，后续不再触发
禁用条件：已选数量=0时按钮置灰不可点击
导入过程：显示导入进度，完成后高亮刚导入的游戏可交互
14"手动选择文件夹"
点击 → 拉起macOS系统文件选择器
用途：自动扫描未找到时，用户手动指定游戏目录
选择后：扫描指定目录，找到游戏则加入列表可交互
页面5：新手秒玩列表
15云游戏列表 + "秒玩"按钮
点击秒玩 → 直接启动该游戏的云游戏
账号模式：以游客模式游玩（无需登录）
数据来源：后台"新手推荐池（云游戏）"专题管理
列表数量：展示最多8款，按热度排序
引导状态：成功启动秒玩 → 标记为已完成可交互
16免费游戏列表 + "免费"按钮
点击免费 → 跳转该游戏详情页，开始下载
账号模式：以游客模式游玩
数据来源：后台"新手推荐池（免费游戏）"专题管理
游戏类型：Steam/GOG平台免费游戏（Demo版、F2P、限免）
验证要求：仅含已验证可在平台正常运行的游戏可交互
17"更多"按钮
云游戏"更多" → 跳转"玩游戏"tab → 云游戏页
免费游戏"更多" → 跳转"玩游戏"tab → PC游戏页
提示：进入云游戏页时顶部提示"恭喜已获得15分钟免费游戏时长"可交互
18新手礼盒弹窗
触发时机：进入新手列表页后自动弹出
内容："已赠送新手礼包 / 获得15分钟免费秒玩时长"
行为：2.5秒后自动消失，无需用户操作
时长说明：此15分钟为账号注册时默认赠送的时长，不额外赠送，弹窗仅作为告知提示
重复进入：仅首次进入展示，重复进入不再弹出自动流程
页面6：探索页（游客态）
19游客头像按钮
点击 → 弹出个人中心面板
位置：左侧导航栏底部
默认头像：系统默认猫头像可交互
20游客个人中心面板
显示内容：
• 游客名称 + "游客模式"金色标签
• 用户ID：-- （游客无ID）
• 登录引导提示："登录/注册后可保存进度、同步数据"
• 云游戏体验卡："新用户注册即送15分钟体验时长" + "去体验"按钮
• 官方社群入口
• 设置入口
• "前往登录"链接

差异点：已登录用户显示真实昵称/ID/充值入口等，游客态隐藏这些游客/登录差异
全局异常
E1接口超时/网络异常
场景：引导页数据接口请求失败
处理：跳过引导，直接进入探索页
重试：下次登录重新触发引导降级
E2中途退出（杀进程/关闭窗口）
场景：用户在引导流程中强制关闭APP
处理：不记录为已完成，下次打开APP重新进入引导页
数据：已选择但未确认的导入游戏不保存
扫描相关异常
E4扫描超时
触发条件：扫描耗时 > 10秒
UI表现：停止动画，弹出提示"扫描超时，请手动选择"
操作：展示"手动选择文件夹"按钮超时
E5扫描结果为空
场景：扫描完成但未找到任何游戏文件
UI表现：展示"未检测到游戏文件"提示
选项：① 手动选择文件夹 ② 去看看其他游戏（跳转探索页）
E6外接硬盘未连接
场景：优先扫描 /Volumes/ 但无外接设备
处理：静默跳过，继续扫描其他路径
不提示：不需要告知用户"未检测到外接硬盘"
导入相关异常
E8导入失败（文件损坏/路径失效）
场景：游戏文件在扫描后被移动/删除
处理：该条标记为导入失败（红色提示），其他正常导入的游戏不受影响
文案："导入失败：文件不存在或已损坏"
E9磁盘空间不足
场景：导入游戏时目标磁盘空间不够
处理：提示"磁盘空间不足，请清理后重试"
按钮状态：导入按钮可用，点击后弹提示
Steam绑定异常
E10Steam登录失败/取消
场景：用户取消Steam登录或网络超时
处理：返回"游玩方式选择页"，不记录为已完成
文案：取消→无提示；失败→Toast"绑定失败，请检查网络后重试"
E11Steam账号已被其他用户绑定
场景：该Steam账号已绑定到平台内另一个用户
处理：提示"该Steam账号已被绑定，请使用其他账号或联系客服"
不可覆盖：不允许强制解绑注意
秒玩相关异常
E12云游戏启动失败
场景：云游戏服务不可用/排队满
处理：提示"当前云游戏繁忙，请稍后再试"
引导状态：启动失败不标记为已完成
按钮恢复："秒玩"按钮恢复可点击状态
E13秒玩时长用完
场景：15分钟免费时长已耗尽
处理：游戏内弹出"体验时长已用完"弹窗
选项：① 充值继续 ② 退出游戏
注意：此场景不在引导页内，在游戏运行时触发
新手列表边界
E14新手推荐池为空
场景：后台未配置推荐游戏池，或接口返回空列表
处理：跳过新手列表，直接进入探索页
降级：不展示空白列表页面降级
E15游戏封面加载失败
场景：CDN图片加载超时/404
处理：展示占位色块 + 游戏名文字
不影响交互：秒玩/免费按钮仍可正常点击
