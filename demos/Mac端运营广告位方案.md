# Mac端运营广告位方案 - 功能改动点

## 背景
Mac端新增运营广告位，用于运营活动、热门游戏、资讯公告、新功能推广等内容推荐。

---

## 新增运营资源位（共4个）

### 1. 启动弹窗（P0 · 冷启动触达）
- **触发时机**：用户打开客户端时弹出
- **形态**：居中模态弹窗，包含活动图片 + 标题 + 描述 + 「立即参与」按钮
- **关闭方式**：弹窗外右上角 X 按钮
- **频控**：支持「今日不再显示」勾选，基于本地存储

### 2. 首页Banner轮播（P0 · 被动曝光）
- **位置**：首页内容区顶部大图
- **形态**：5张全屏轮播图，6秒自动切换
- **交互**：左右箭头 + 底部导航圆点
- **内容**：运营活动、热门游戏推荐、平台公告均可配置

### 3. 编辑推荐运营插卡（P1 · 自然融入）
- **位置**：首页「编辑推荐」卡片列表第3位
- **形态**：与普通游戏卡片同尺寸，紫色边框 + 左上角「限时活动」标签区分
- **无额外操作按钮**，点击整卡跳转

### 4. 活动中心页（做法2 · 主动回访入口）
- **入口**：左侧导航栏新增「活动中心」icon，带红点提示未读
- **页面**：一排2个活动卡片，左侧图片 + 右侧文案（标签/标题/描述/时间）
- **内容**：进行中活动、公告、新功能上线等

### 5. 游戏库常驻运营位（做法3 · 场景伴随）
- **位置**：游戏库页面右侧边栏顶部
- **形态**：活动图 + 标签 + 标题 + 描述，常驻展示
- **下方**：Steam平台信息卡（在线状态、游戏数量、时长等）

---

## 运营触达链路

```
冷启动 → 启动弹窗（一次性强触达）
       ↓ 关闭后
日常浏览 → Banner轮播（被动曝光）+ 编辑推荐插卡（自然融入）
       ↓ 想回看时
主动查看 → 左侧导航「活动中心」（随时可达）
       ↓ 玩游戏时
场景伴随 → 游戏库右侧常驻运营位（持续露出）
```
