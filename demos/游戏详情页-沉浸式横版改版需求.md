# 游戏详情页 · 沉浸式横版改版需求

## 一、背景与目标

当前游戏详情页采用「左侧滚动内容 + 右侧固定信息面板」的双栏布局，用户进入页面后感知到的是「一个 App 界面」而非「一款游戏」。

**改版目标：** 以游戏 Key Art 作为首屏主视觉，信息元素悬浮于画面之上，提升沉浸感与游戏氛围感，同时保留完整的功能信息。

---

## 二、页面结构概览

页面分为两屏，在横版手机内垂直滚动，支持吸附切换。

| 屏次 | 内容 |
|------|------|
| 首屏 | 游戏沉浸封面 + 核心信息 + 底部数据栏 |
| 次屏 | 视频/图集 + 游戏介绍图文 |

---

## 三、首屏

### 3.1 顶部导航

- 左上角：返回按钮
- 右上角：系统状态栏（信号 / WiFi / 电量 / 时间）

### 3.2 主内容区

位于首屏左侧，内容从上到下依次为：

1. **游戏标题**
2. **副标题** — 一句话描述，如「生存、建造、射击，只不过你是只鸭子」
3. **标签行** — 从左到右依次为：
   - **平台切换胶囊**（Steam / Epic Games，单选，默认选中 Steam）
   - 热门榜标签（如「🔥 热门榜 TOP 1」）
   - 游戏分类标签（如「生存开放世界」「多人联机」），支持换行
4. **游戏简介** — 最多显示 2 行，超出省略
5. **启动方式按钮组** — 从左到右：秒玩 → 获取游戏 → 更多菜单（九宫格）

**更多菜单（九宫格点击展开）包含：**
- PC游戏设置
- 添加到桌面
- 卸载游戏

### 3.3 底部数据栏

固定在首屏底部，**4列 × 2行**网格，每项上方为字段名、下方为数值。

**第 1 行（优先展示）：**

| 兼容性评价 | 总游戏时长 | 云存档 | 玩家评价 |
|-----------|-----------|--------|---------|
| ★★★★☆ + 向右箭头（可点击跳转兼容性详情页） | 139.3h | 已是最新 | 9.5 |

**第 2 行：**

| 年龄评级 | 游戏语言 | 游戏团队 | 发布日期 |
|---------|---------|---------|---------|
| 6+ | 中文 8+ | Team Soda | Oct 16, 2025 |

## 四、次屏

### 4.1 视频 / 图集区

- 默认播放游戏宣传视频（自动播放、静音、循环）
- 底部控制层包含：
  - 进度条（可点击跳转）
  - 视频 / 图集切换
  - 静音切换、全屏按钮

### 4.2 游戏介绍区

内容块顺序：

1. **游戏介绍** — 文字段落
2. **关于游戏** — 文字段落
3. **游戏视频** — 支持用户控制播放
4. 内容结束标识

---

## 五、交互说明

| 触发 | 行为 |
|------|------|
| 首屏向上滑动 | 切换至次屏（吸附） |
| 次屏向下滑动 | 返回首屏（吸附） |
| 点击平台切换（Steam/Epic） | 切换当前游戏来源平台，高亮选中项，榜单数据随平台更新 |
| 点击秒玩 | 触发秒玩启动流程 |
| 点击获取游戏 | 触发游戏获取/下载流程 |
| 点击九宫格 | 展开更多操作菜单（浮层） |
| 点击菜单外区域 | 关闭更多菜单 |
| 点击视频进度条 | 跳转到对应播放位置 |
| 点击视频/图集切换 | 切换媒体展示形式 |
| 点击静音按钮 | 切换视频静音状态 |
| 点击返回 | 返回上一页 |

## 六、数据说明

### 底部数据栏字段来源

| 字段 | 数据来源 |
|------|---------|
| 兼容性评价 | 游戏兼容性检测结果（星级，满分5星，可点击跳转详情） |
| 总游戏时长 | 当前用户游玩时长 |
| 云存档 | 云存档同步状态 |
| 玩家评价 | 游戏评分聚合（0-10分值） |
| 年龄评级 | 游戏基础信息 |
| 游戏语言 | 游戏基础信息 |
| 游戏团队 | 游戏基础信息（开发商） |
| 发布日期 | 游戏基础信息 |

### 标签数据来源

| 标签类型 | 数据来源 |
|---------|---------|
| 热门榜 TOP N | 实时榜单排名 |
| 游戏分类标签 | 游戏基础信息（标签列表） |
