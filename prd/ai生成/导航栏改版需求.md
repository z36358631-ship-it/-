# 【PRD】《盖世游戏》底部导航栏改版需求

| 版本 | 策划人 | 日期 | 修订内容 |
|------|--------|------|---------|
| V1.0 | 郑群超 | 2026.05.25 | 初版，合并首页+玩游戏，新增社区入口 |

---

## 1. 功能概述

### 1-1 功能描述

合并底部导航"探索"和"玩游戏"为统一"首页"，腾出tab位给"社区"入口。改版后导航结构支持社区、游戏库、Steam全链路等后续功能扩展。

### 1-2 功能结构

```
改版前（5tab）：
  探索 | 玩游戏 | 游戏库 | 排行 | 我的

改版后（5tab）：
  首页 | 社区 | 游戏库 | 排行 | 我的
```

### 1-3 名词注释

| 名词 | 说明 |
|------|------|
| 首页 | 合并原"探索"和"玩游戏"内容，作为APP默认落地页 |
| 社区 | 新增tab，承载游戏圈、帖子动态、创作者中心等社区功能 |
| 游戏库 | 保持不变，Steam/Epic/本地导入游戏管理 |

---

## 2. 界面说明

### 2-1 底部导航栏

#### ① 竖屏模式

底部固定导航栏，5个tab等宽排列。

| 位置 | tab名称 | icon | 默认选中 | 说明 |
|------|---------|------|---------|------|
| 1 | 首页 | 🏠 (house) | ✅ 是 | APP启动后默认落地页 |
| 2 | 社区 | 💬 (message-square) | | 点击后跳转至社区首页 |
| 3 | 游戏库 | 🎮 (gamepad) | | 保持现有功能不变 |
| 4 | 排行 | 🏆 (trophy) | | 保持现有功能不变 |
| 5 | 我的 | 👤 (user) | | 保持现有功能，后续增加创作者中心入口 |

(1) 选中态：icon+文字高亮为主色(#D4B473)，未选中态为灰色(#8E8E93)
(2) 社区tab右上角：若有未读消息，显示红点角标
(3) 切换tab时无页面跳转，tab内容区域切换，保留滚动位置

#### ② 横屏模式（侧边栏）

左侧固定侧边栏，宽80px，垂直排列5个tab。

(1) 选中态：左侧显示3px高亮竖条(#D4B473)，icon放大至28px
(2) 其余逻辑同竖屏

### 2-2 首页（合并后）

原"探索"和"玩游戏"合并为"首页"，顶部使用横向tab切换子板块。

#### ① 首页顶部tab

```
首页顶部：
  [搜索框]
  推荐 | PC游戏 | 云游戏 | 复古 | 资讯
```

| tab | 原归属 | 内容说明 |
|-----|--------|---------|
| 推荐 | 探索页 | Banner + 为你推荐 + 热门游戏卡片 |
| PC游戏 | 玩游戏 | Steam/Epic/本地导入游戏列表，原"玩游戏"主要内容 |
| 云游戏 | 玩游戏 | 云游戏列表+秒玩入口 |
| 复古 | 玩游戏 | 复古游戏列表 |
| 资讯 | 探索页 | 最新资讯/公告 |

(1) tab支持左右滑动，超出屏幕宽度时可滚动
(2) 默认选中"推荐"
(3) 下拉刷新，上拉加载更多

#### ② 搜索框

位于首页顶部，固定位置。

(1) 点击后跳转至搜索页
(2) 搜索框内显示placeholder热词，由后台配置
(3) 搜索范围覆盖：游戏名 / 社区帖子 / 用户

### 2-3 社区tab（新增）

点击"社区"tab后显示社区首页。详细界面设计见 @郑群超 社区游戏圈需求文档。

简要结构：
```
社区首页：
  顶部tab：推荐 | 关注 | 游戏圈
  内容区：帖子瀑布流 / 游戏圈列表
  FAB：发帖按钮
```

### 2-4 游戏库 / 排行 / 我的

保持现有功能不变。

"我的"页面后续迭代新增：
- 创作者中心入口
- 存档管理入口
- 积分/等级展示

### 2-5 状态与异常处理

| 状态 | 处理 |
|------|------|
| 首次安装打开 | 默认选中"首页"→"推荐"子tab |
| 从游戏详情页返回 | 返回至离开时的tab和滚动位置 |
| 网络异常 | 各tab内容区显示"网络异常，点击重试" |
| 社区未开放（灰度） | 社区tab置灰，点击弹toast"即将上线" |

## 3. 数据埋点

| 页面名称 | 事件类型 | 功能名称 | 数据统计 | 上报字段 |
|---------|---------|---------|---------|---------|
| 底部导航栏 | 点击事件 | tab切换 | 人数、人次 | tab_name, from_tab |
| 首页 | 浏览事件 | 首页展示 | 人数、人次 | sub_tab(推荐/PC游戏/云游戏/复古/资讯) |
| 首页 | 点击事件 | 子tab切换 | 人数、人次 | sub_tab_name, from_sub_tab |
| 首页 | 点击事件 | 搜索框点击 | 人数、人次 | — |
| 社区 | 浏览事件 | 社区首页展示 | 人数、人次 | sub_tab(推荐/关注/游戏圈) |
| 社区 | 点击事件 | 社区tab红点清除 | 人数、人次 | unread_count |

PS：以上所有数据上报都必须包括用户公共属性（见数据上报列表）
