# 盖世游戏 · 社区（游戏圈）功能方案

---

## 1. 功能目标

**让每个用户踩过的坑变成下一个用户的路。**

量化目标：
- 新用户通过社区帖子找到可用配置的比例 > 30%
- 社区DAU占APP DAU的 40%+
- 每个Top 50游戏的游戏圈至少有20条有效配置帖

---

## 2. 核心功能说明

### 必须有（本期）

| 功能 | 解决什么问题 | 对应用户故事 |
|------|------------|------------|
| **游戏圈** | 每个游戏一个独立讨论区，帖子按游戏聚合不散落 | 故事1：找同游戏同机型的配置 |
| **配置帖模板** | 发配置帖时自动带入机型/Wine版本/参数，降低分享门槛 | 故事2：分享配置不用手打一堆参数 |
| **帖子分类** | 攻略/配置/求助/存档分开，找东西不用翻 | 故事1、3：快速定位需要的内容 |
| **评论/楼中楼** | 追问细节、补充信息 | 故事1：问"你用的什么DXVK版本" |
| **推荐流** | 不知道看什么时有内容刷 | 故事4：围观发现新游戏 |
| **发帖** | 图文帖+关联游戏+分类标签 | 故事2、5：分享内容 |
| **搜索** | 按游戏名/关键词找帖子 | 故事1：搜"骁龙8gen3 GTA5 配置" |

### 后续再做

| 功能 | 为什么不急 |
|------|-----------|
| 创作者等级/收益 | 先有内容再谈激励，冷启动阶段靠运营驱动 |
| 积分商城 | 依赖商品供给（CDK/云游戏时长），需要商业化体系配合 |
| 视频帖 | 图文够用，视频开发成本高且审核复杂 |
| MOD一键安装 | 技术复杂，先做分享链接/文件下载 |
| 存档一键导入 | 依赖存档路径配置库，先做手动下载 |
| 话题/热榜运营 | 先跑通基础发帖-浏览闭环 |

## 3. 用户主流程

### 流程A：求助者（占50%用户）

```
打开APP → 底部tab"社区" → 搜索"GTA5 骁龙8gen3"
  → 看到3条配置帖 → 点进去看参数
  → 评论追问"分辨率选多少" → 得到回复
  → 按帖子配置启动游戏 → 成功
```

### 流程B：分享者（占15%用户）

```
游戏跑通了 → 点FAB发帖 → 自动带入机型+Wine版本
  → 选择游戏"GTA5" → 选分类"配置"
  → 写正文补充说明 → 贴截图
  → 发布 → 收到点赞通知
```

### 流程C：围观者（占25%用户）

```
打开APP → 底部tab"社区" → 刷推荐流
  → 看到"空洞骑士手柄完美映射方案"帖子
  → 点赞+收藏 → 想玩这个游戏了
  → 点帖子里的游戏名 → 跳转游戏详情页
```

### 流程D：存档玩家（占5%用户）

```
进入星露谷物语游戏圈 → 切到"存档"tab
  → 看到"全收集完美存档" → 点进去
  → 下载附件(zip) → 手动放入存档目录
  （后续迭代：一键导入）
```

## 5. 用户可操作动作

### 浏览侧

| 动作 | 触发位置 | 说明 |
|------|---------|------|
| 刷推荐流 | 社区首页-推荐tab | 下拉刷新、上拉加载 |
| 进入游戏圈 | 社区首页-游戏圈tab / 游戏详情-社区tab | 查看单个游戏的所有帖子 |
| 切换分类 | 游戏圈详情页sub-tab | 综合/精选/攻略/配置/求助/存档 |
| 搜索 | 社区首页搜索框 | 按关键词搜帖子/游戏圈/用户 |
| 查看帖子详情 | 点击帖子卡片 | 全文+评论 |

### 互动侧

| 动作 | 触发位置 | 说明 |
|------|---------|------|
| 点赞 | 帖子卡片/详情页 | 即时反馈，计入作者积分 |
| 收藏 | 帖子详情页 | 收藏到"我的收藏" |
| 评论 | 帖子详情页底部 | 支持楼中楼回复 |
| 关注游戏圈 | 游戏圈详情页 | 关注后内容出现在"关注"tab |
| 关注用户 | 用户主页 | 关注后内容出现在"关注"tab |
| 举报 | 帖子/评论长按 | 选择原因提交 |

### 创作侧

| 动作 | 触发位置 | 说明 |
|------|---------|------|
| 发帖 | FAB按钮 | 必须关联游戏+选分类 |
| 发配置帖 | 发帖页选"配置"分类 | 自动带入设备+Wine信息 |
| 发求助帖 | 发帖页选"求助"分类 | 自动带入设备信息+错误日志（若有） |
| 上传存档附件 | 发帖页选"存档"分类 | 支持zip/rar，≤500MB |
| 编辑/删除自己的帖子 | 帖子详情-更多菜单 | |

### 导航侧

| 动作 | 触发位置 | 说明 |
|------|---------|------|
| 从帖子跳转游戏详情 | 帖子内游戏名标签 | 点击游戏名→游戏详情页 |
| 从游戏详情进社区 | 游戏详情页-社区tab | 查看该游戏的游戏圈 |
| 从社区进游戏圈 | 社区首页-游戏圈列表 | 点击游戏圈卡片 |

## 6. 非目标范围（本期不做）

| 不做的事 | 原因 |
|---------|------|
| 视频帖 | 开发成本高+审核复杂，图文已够用。用户想发视频可以发B站链接 |
| 积分商城 | 需要商品供给体系（CDK/时长），先跑通内容闭环再接入 |
| 创作者收益提现 | 需要财务/税务/合规体系，先用积分+虚拟奖励替代 |
| MOD一键安装 | 技术复杂（需要注入Wine Prefix），先做文件下载+手动安装教程 |
| 存档一键导入 | 需要维护每个游戏的存档路径配置库，先做文件下载 |
| 话题/热榜运营 | 冷启动阶段帖子量不够撑热榜，先靠运营手动置顶精选 |
| 私信 | 评论区已够沟通，私信带来审核和骚扰风险 |
| 官号自动转发Steam公告 | 需要对接Steam API+内容格式化，先手动运营 |
| 发行人计划（B站/抖音视频奖励） | 需要审核流程+财务结算，先做创作者标识和数据看板 |
| 国际化/多语言社区 | 先做中文社区跑通，海外用户后续单独规划 |

### 本期做和不做的判断标准

```
做：能让"求助者找到答案"和"分享者发出内容"这个闭环跑通的最小功能集
不做：锦上添花的运营工具、复杂的技术集成、需要其他系统配合的功能
```
