# 【Prd】《盖世游戏》手柄壁纸自定义切换需求

| 版本 | 策划人 | 日期 | 修订内容 |
|------|--------|------|----------|
| V1.0 | - | 2026-06-12 | 初版 |

## 1. 功能概述

### 1.1 功能描述

手柄屏幕壁纸自定义功能。用户可通过APP上传图片、裁切后同步至手柄屏幕（分辨率160×80），支持历史记录快速切换和恢复默认。

### 1.2 功能结构

```
壁纸设置页
├── 壁纸预览区（点击拉起相册）
├── 恢复默认（右上角，二次确认后直接生效）
├── 历史记录（最多5张，按时间倒序，不可删除）
├── 保存壁纸（底部按钮）
└── 裁切页（长方形裁切框，支持缩放/平移）
```

### 1.3 名词注释

| 名词 | 说明 |
|------|------|
| 壁纸 | 手柄小屏幕上显示的图片，分辨率160×80 |
| 裁切框 | 固定2:1比例的长方形选区 |
| 历史记录 | 用户曾经裁切/上传过的壁纸缓存 |

## 2. 界面说明

### 2.1 壁纸设置页

#### ① 导航栏

- 左侧返回按钮，点击后返回上一页
- 标题：壁纸设置
- 右侧"恢复默认"文字按钮

#### ② 壁纸预览区

- 初始状态（未设置壁纸）：居中显示加号icon + 文案"请选择壁纸"，无"点击更换"提示
- 已有壁纸时：展示当前壁纸大图，右下角显示"点击更换"提示icon
- 点击后拉起系统相册，选择图片后进入裁切页

#### ③ 历史记录区

- 标题：历史记录
- 竖屏：1行2张，横屏：1行4张
- 每张图右下角显示时间标签（格式：M/D H:mm）
- 选中态：蓝色边框 + 右上角蓝色勾选标记
- 点击历史记录图片 → 选中该壁纸，预览区同步更新
- 历史记录不可手动删除

#### ④ 保存壁纸按钮

- 底部固定，圆角蓝色按钮
- 未选中壁纸时置灰不可点击
- 点击后壁纸数据通过蓝牙同步至手柄
- 操作成功提示toast："保存成功"

### 2.2 横屏布局

- 左右分栏布局
- 左侧：壁纸预览区（居中展示）
- 右侧：历史记录列表（1行4张）
- 底部保存按钮通栏

### 2.3 裁切页

- 全屏黑色背景覆盖
- 顶部：左侧"取消"、右侧"确定"
- 中部：图片展示区，固定2:1长方形裁切框（白色边框），框外半透明黑色遮罩
- 底部提示文案："拖动和缩放图片，调整壁纸显示区域"
- 交互：
  - 单指/鼠标拖拽：平移图片
  - 双指捏合/鼠标滚轮：缩放图片
  - 图片不可移出裁切框边界（始终填满裁切框）
  - 缩放范围：1倍～4倍（基于填满裁切框的最小缩放）
- 点击"确定" → 裁切完成，壁纸直接生效为当前壁纸，同时写入历史记录

### 2.4 恢复默认

- 入口：导航栏右上角"恢复默认"
- 点击后弹出二次确认弹窗：
  - 文案第一行："确定恢复默认壁纸吗？"
  - 文案第二行（灰色小字）："恢复后请通过手柄查看壁纸"
  - 按钮：取消 / 确定
- 点击"确定" → 壁纸设置页无任何变化，手柄端直接恢复为默认壁纸
- toast提示："已恢复默认壁纸，请通过手柄查看"

## 3. 逻辑规则

### 3.1 历史记录

- 最多保存5张
- 每次裁切成功自动写入历史记录
- 若已满5张，自动替换时间最早的一条
- 按时间倒序排列（最新在前）
- 历史记录不可手动删除
- 从历史记录选择壁纸并保存时，更新该条时间戳至当前

### 3.2 壁纸同步

- 壁纸数据需在手柄蓝牙连接状态下同步
- 输出分辨率：160×80
- 文件格式：无限制
- 文件大小：无限制
- 图片数据格式：由裁切区域原始像素输出

### 3.3 状态流转

| 状态 | 预览区展示 | 保存按钮 |
|------|-----------|----------|
| 初次进入（未设置） | "请选择壁纸" | 置灰不可点 |
| 已选择/已裁切 | 当前壁纸大图 | 可点击 |
| 恢复默认后 | 默认壁纸 | 可点击 |

### 3.4 操作反馈

| 操作 | 反馈 |
|------|------|
| 保存壁纸成功 | toast "保存成功" |
| 恢复默认成功 | toast "已恢复默认壁纸" |
| 裁切确定 | 无toast，直接生效 |

## 4. 图片上传规则

- 支持格式：JPG/PNG/GIF/WebP等系统相册可识别的格式
- 单张大小：无限制
- 裁切比例：固定2:1（160×80）
- 裁切后展示：使用原始裁切区域像素大图展示（非压缩至160×80的小图）
- 裁切后输出：同步至手柄时压缩为160×80

## 5. 异常场景

| 场景 | 处理 |
|------|------|
| 蓝牙未连接 | 允许裁切和选择，保存时提示需连接手柄 |
| 图片加载失败 | 展示默认占位 |
| 历史记录为空 | 显示"暂无历史记录" |
