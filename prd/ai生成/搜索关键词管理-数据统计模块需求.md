## 一、功能目的

在游戏关键词管理页顶部增加数据统计卡片区域，运营可直观了解关键词命中效果和点击转化情况，辅助判断关键词运营质量。

---

## 二、功能说明

### 2-1 统计卡片区

位置：游戏关键词管理页顶部，Tab列表上方。

展示5个统计卡片，横向一行排列：

| 统计项 | 定义 | 计算规则 |
|--------|------|----------|
| 关键词总数 | 当前已生效关键词总数 | 运营手动添加 + AI审核通过，底部标注"运营 X / AI X" |
| 命中次数 | 关键词被搜索命中的总次数 | 用户搜索词匹配到关键词库的请求次数累计 |
| 命中占比 | 关键词命中占全部搜索的比例 | 命中次数 / 全部搜索请求总次数（含无结果请求）×100% |
| 点击量 | 通过关键词召回的游戏被点击总次数 | 搜索结果中因关键词匹配召回的游戏，用户点击进入详情的次数 |
| 点击占比 | 关键词召回点击占搜索总点击比例 | 关键词召回点击量 / 所有搜索结果总点击量 ×100% |

RS：关键词总数不受时间筛选影响，始终显示当前实时数据；其余4项受时间筛选控制。

### 2-2 筛选条件

筛选区位于Tab下方，表格上方，一行展示：

| 筛选项 | 类型 | 说明 |
|--------|------|------|
| 搜索词/命中词 | 文本输入 | 模糊匹配搜索词或推荐命中词 |
| 类型 | 下拉单选 | 简称/俗称·黑话/拼音·缩写/错别字/角色·IP |
| 关联游戏 | 文本输入 | 模糊匹配推荐关联游戏名称 |
| 审核人 | 文本输入 | 精确匹配审核人姓名 |
| 时间 | 快捷+自定义 | 快捷按钮：昨日、近7天、近30天；支持自定义起止日期 |

末尾提供"查询"和"重置"按钮：
- 点击"查询"→ 按条件重新加载统计卡片数据及表格数据
- 点击"重置"→ 清空所有筛选条件，恢复默认（近7天）

RS：时间筛选同时作用于顶部统计卡片和下方表格列表。

### 2-3 关键词列表

Tab分组：全部 / 待审核 / 已生效 / 已忽略

表格字段：

| 字段 | 说明 |
|------|------|
| 搜索词 | 用户原始输入的搜索文本 |
| 推荐命中词 | 系统标准化/纠正后的关键词 |
| 命中词类型 | 简称/俗称/拼音/错别字/IP；AI未识别时为空 |
| 推荐关联游戏 | AI推荐的匹配游戏及可信度；未识别时为空，可人工编辑补充 |
| 搜索次数 | 该关键词累计搜索次数，支持升降序排序 |
| 搜索人数 | 搜索该词的去重用户数，支持升降序排序 |
| 处理状态 | 待AI分析/待审核/已通过/已忽略 |
| 审核人 | 最终处理的运营人员 |
| 审核时间 | 人工审核通过/忽略的时间 |
| 操作 | 见2-4 |

### 2-4 操作按钮

按处理状态区分操作项：

| 状态 | 可用操作 |
|------|----------|
| 待AI分析 / 待审核 | 编辑、通过、修改后通过、忽略 |
| 已通过 | 编辑、删除 |
| 已忽略 | 编辑、删除 |

操作说明：

(1) **编辑**：弹窗修改推荐命中词、推荐关联游戏、命中词类型

(2) **通过**：二次确认弹窗"确认通过'XX'→'关联游戏'？"；确认后状态变为已通过，记录审核人和审核时间，关键词进入正式关键词库

(3) **修改后通过**：打开编辑弹窗，修改关联游戏/命中词后保存，状态变为已通过

(4) **忽略**：二次确认弹窗"忽略后将清空搜索次数记录，不再进入审核列表"；确认后状态变为已忽略，搜索次数清零

(5) **删除**：二次确认弹窗"删除后不可恢复"；确认后从关键词库移除该条记录

### 2-5 批量操作

批量操作按钮位于筛选条件同行右侧：

| 按钮 | 说明 |
|------|------|
| 批量通过 | 勾选多条待审核记录后，二次确认，批量变更为已通过 |
| 批量忽略 | 勾选多条记录后，二次确认，批量变更为已忽略并清零搜索次数 |
| 批量导入 | 弹窗上传Excel/CSV文件，模板字段：游戏ID、游戏名称、关键词（多个用英文逗号分隔） |
| 新增关键词 | 弹窗新增，字段：搜索词、推荐命中词、关联游戏ID、游戏名称（选填）、命中词类型 |

### 2-6 新增关键词弹窗

| 字段 | 是否必填 | 说明 |
|------|----------|------|
| 搜索词 | 是 | 用户实际搜索的原始文本 |
| 推荐命中词 | 是 | 纠正/归一后的标准关键词 |
| 关联游戏ID | 是 | 优先使用游戏ID匹配 |
| 游戏名称 | 否 | 人工确认用 |
| 命中词类型 | 否 | 下拉选择：简称/俗称·黑话/拼音·缩写/错别字/角色·IP |

## 三、异常处理

| 场景 | 处理 |
|------|------|
| 筛选时间范围内无数据 | 统计卡片显示0，表格显示空状态"暂无数据" |
| 批量通过时所选无待审核项 | 弹出提示"所选项中无待审核记录" |
| 批量导入文件格式错误 | 弹窗展示失败原因，支持下载失败明细 |
| 新增关键词必填项为空 | 弹出提示"请填写必填项" |
| 删除已通过关键词 | 该关键词从正式库移除，用户搜索时不再召回对应游戏 |
