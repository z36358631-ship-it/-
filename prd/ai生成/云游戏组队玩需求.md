## 核心入口

- 默认进入云游戏首页，保留原有云游戏秒玩链路。
- 顶部新增「组队玩」入口，点击后进入独立的组队玩法页面。
- 云游戏首页在「最近常玩」下方展示「组队热玩」专题。
- 「组队热玩」标题右侧展示「查看全部」，与标题同一行右对齐，点击跳转到组队玩页的「游戏大厅」。

## 核心页面

- 云游戏首页：展示用户信息、最近常玩、组队热玩、秒玩人气热游等模块。
- 组队玩页：包含「游戏大厅」「组队大厅」「世界频道」三个子 Tab，从左到右排列。
- 游戏大厅：按游戏展示可组队游戏，显示房间数；有房间时按钮为「快速加入」，无房间时按钮为「创建房间」。同一行右侧展示刷新按钮、举报按钮和「玩法说明」入口，不在组队大厅展示玩法说明。
- 组队大厅：保持原有卡片列表样式，按房间展示公开房间，支持搜索房间名、房间 ID、游戏名；右下角悬浮创建房间入口。横屏下一排 2 个房间卡片。 //2026.5.15修改
- 世界频道：与「游戏大厅」「组队大厅」同级切换，支持查看频道消息和快捷发言，房主可通过「世界招募」按钮同步发送房间分享到世界频道。 //2026.5.15修改
- 玩法说明：仅在游戏大厅同一行右侧展示，点击进入独立说明页。
- 组队房间：独立页面，左上角返回，展示游戏名、房间 ID、成员席位和底部操作按钮。

## 核心链路

- 首页组队热玩 -> 快速加入 / 创建房间 -> 进入组队房间 -> 开始云游戏。
- 组队玩 -> 游戏大厅 -> 选择游戏 -> 快速加入 / 创建房间 -> 进入组队房间。
- 组队玩 -> 组队大厅 -> 搜索或浏览房间 -> 加入房间 -> 等待房主开始。
- 房主离开房间页后，在 App 容器内展示房间悬浮条，支持返回房间或退出房间。

## 游戏内与异常

- 游戏内为 App 内全屏云游戏界面，横版 860x400，竖版 400x860。
- 游戏内展示 P1-P4 队伍状态，支持在线 / 掉线状态展示。
- 游戏内默认开启麦克风和听筒，支持手动关闭。
- 房主或队友掉线后，重新启动 App 时展示重连弹窗，支持「稍后再说」和「重连继续」。

## 分享与房间信息

- 房间 ID 支持点击复制，复制成功后 toast 提示。
- 房间页右上角「...」打开更多菜单，包含修改房间信息等操作。 //2026.5.15修改
- 分享弹窗仅展示微信、QQ、复制链接三个入口。
- 分享房间需要 10 秒一次 CD。
- <font color="#4e73ff">**「分享房间」入口改为「世界招募」胶囊按钮，放置在房间页消息输入框右侧，不再收进「...」菜单。房主点击后将房间信息同步发送至世界频道，10 秒 CD 内按钮置灰。**</font> //2026.5.15修改

## 时长与状态限制

- 加入房间、快速加入和创建房间前需校验云游戏时长。
- 时长不足时展示充值引导弹窗，弹窗文案居中，左侧「稍后再说」，右侧「立即充值」。
- 房间人数不足时，「开始游戏」置灰不可用。

## 组队房间页

- 页面标题为「组队房间」，左上角返回按钮。
- 顶栏右侧包含语聊控制按钮（麦克风、听筒），以及「...」更多菜单。
- 房间页首屏无大块空白，核心内容可见，首屏能看到「开始游戏」按钮。
- 房间 ID 为 6 位随机数，跟在游戏名后面，不需要背景框，支持点击复制，复制成功 toast 提示。
- 成员席位布局：横屏房主 + 队友一排 4 个；竖屏可压缩为一排 2 个。
- 移除「已准备」状态，按钮为「解散房间」和「开始游戏」。
- <font color="#4e73ff">**移除房间页独立的「房间语聊」区块，语聊控制按钮（麦克风 / 听筒开关）移至顶栏右侧。**</font> //2026.5.15修改
- <font color="#4e73ff">**移除房间页独立的「队伍须知」区块，相关内容整合到玩法说明第 5 条。**</font> //2026.5.15修改
- <font color="#4e73ff">**字体全面缩小以适配横竖屏显示，确保核心内容在横屏小屏幕下不截断。**</font> //2026.5.15修改

## 统一聊天流框架（chat-flow）

房间消息和世界频道共用同一套聊天流框架，规则如下：

- <font color="#4e73ff">**消息布局：自己发送的消息靠右显示，头像在右侧；其他人发送的消息靠左显示，头像在左侧。**</font> //2026.5.15修改
- 消息气泡包含：头像 + 昵称 + 文本气泡 / 卡片。
- <font color="#4e73ff">**时间戳格式为 `2026/5/15 21:07`，每 3 分钟间隔显示一次，居中展示于消息流中间。**</font> //2026.5.15修改
- 快捷短语点击后随机切换为另一条短语，短语内容由后台配置。 //2026.5.15修改

### 世界频道游戏卡片

- <font color="#4e73ff">**世界频道中的房间分享展示为游戏卡片，卡片内容：游戏封面 + 游戏名 + 房间名 + 房间 ID + 状态标签（准备中 / 游戏中）+ 「立即加入」按钮。**</font> //2026.5.15修改

## 举报

- <font color="#4e73ff">**举报按钮位于游戏大厅刷新按钮旁边（同一行）。**</font> //2026.5.15修改
- 点击举报按钮弹出 toast 提示：「不良信息举报请截图发送至 gamehub@gamesir.com」。

## 快捷短语

- 房间消息和世界频道均支持快捷发言。
- <font color="#4e73ff">**快捷短语点击后随机切换为另一条短语，短语内容由后台配置，每次点击展示的短语不与上一条相同。**</font> //2026.5.15修改

## 创建房间

- 创建房间弹窗不能超出容器。
- 游戏选择需有 icon / 封面辅助识别。
- 私密房间需设置密码。
- 创建成功后进入组队房间。
- 仅离开房间页去其他页面时才展示房间悬浮条。
- <font color="#4e73ff">**房间名最多输入 20 个字，默认名称为「来不及解释了，快加入房间！」（共 14 字，在限制范围内）。**</font> //2026.5.15修改

## 弹窗统一规范

- <font color="#4e73ff">**所有弹窗（创建房间、充值引导、解散房间确认、退出房间确认等）统一宽度为 300px，标题居中显示，内容区域采用紧凑布局，减少无效留白。**</font> //2026.5.15修改

## 房间悬浮条

- 悬浮条在容器内，不超出容器边界。
- 展示游戏 icon、游戏名、准备中状态、有效期。
- 支持「进入房间」和「退出房间」；退出需二次确认。

## Mac 端新增模块

- <font color="#4e73ff">**Mac 端探索页在「最新资讯」上方新增「组队热玩」专题，展示可组队的热门游戏卡片列表。**</font> //2026.5.15修改
- <font color="#4e73ff">**Mac 端秒玩页新增「组队玩」Tab，Tab 下包含游戏大厅、组队大厅、世界频道三个子切换入口，与移动端逻辑一致。**</font> //2026.5.15修改

## 横竖屏适配

- 支持横竖屏切换，横屏参考横版玩游戏 Demo 布局。
- 竖版右上角保留横竖屏切换按钮。
- 游戏内横版尺寸 860x400，竖版 400x860。
