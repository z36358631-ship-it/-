## 1. 功能概述

### 1-1 功能描述

新增"社区"模块，核心为**游戏圈**——每个游戏拥有独立的讨论社区。用户可在游戏圈内发帖（攻略/配置/存档/MOD/求助），形成互助生态。同步上线创作者计划和积分激励体系，驱动优质内容生产。

参照竞品：233乐园游戏圈 + 小黑盒社区

### 1-2 功能结构

```
社区
├── 社区首页（底部tab入口）
│   ├── 推荐（算法推荐帖子）
│   ├── 关注（已关注用户/游戏圈动态）
│   └── 游戏圈（热门游戏圈列表）
├── 游戏圈详情页（单个游戏的社区空间）
│   ├── 综合 / 精选 / 攻略 / 配置 / 存档 / MOD
│   └── 发帖入口
├── 游戏详情页-社区tab（与游戏圈详情联动）
├── 发帖/编辑器
├── 帖子详情页（评论/楼中楼）
├── 创作者中心（我的页面入口）
│   ├── 发行人计划
│   ├── 创作数据看板
│   └── 积分/等级
└── 积分商城
```

## 2. 界面说明

### 2-1 社区首页

入口：底部导航栏"社区"tab

#### ① 顶部区域

搜索框 + tab栏

```
[🔍 搜索游戏圈、帖子、用户]
推荐 | 关注 | 游戏圈
```

(1) 搜索框：点击后跳转至搜索页，支持搜索游戏圈名称、帖子关键词、用户昵称
(2) tab栏：3个tab，左对齐，选中态加粗+底部指示条

#### ② 推荐tab

算法推荐帖子，双列瀑布流布局。

帖子卡片元素：

| 元素 | 说明 |
|------|------|
| 封面图 | 帖子第一张图，若无图则显示纯文字卡片 |
| 标题 | 最多2行，超出截断 |
| 用户头像+昵称 | 左下角 |
| 点赞数 | 右下角，含icon |
| 游戏圈标签 | 左上角角标，显示关联游戏名 |

(1) 下拉刷新
(2) 上拉加载更多（每次20条）
(3) 点击卡片→跳转至帖子详情页

#### ③ 关注tab

已关注用户和游戏圈的帖子，单列时间线布局。

(1) 若未关注任何内容，显示空态："还没有关注，去游戏圈看看 →"
(2) 帖子卡片同推荐tab，但为单列宽卡片

#### ④ 游戏圈tab

热门游戏圈列表，按热度（帖子数+活跃用户数）排序。

游戏圈卡片元素：

| 元素 | 说明 |
|------|------|
| 游戏封面 | 圆角矩形，60×60 |
| 游戏名称 | 主标题 |
| 关注人数 | 副标题，如"12.3万人关注" |
| 今日帖子数 | 副标题，如"今日 328 条新帖" |
| 关注按钮 | 右侧，已关注/未关注两态 |

(1) 顶部：我关注的游戏圈（横向滚动，圆形icon列表）
(2) 下方：热门游戏圈列表（纵向）
(3) 游戏圈排序由后台配置，支持置顶/推荐位
(4) 点击游戏圈卡片→跳转至游戏圈详情页

#### ⑤ FAB发帖按钮

右下角悬浮，金色圆形按钮(#D4B473)，"+"icon。

(1) 点击后弹出发帖页
(2) 若未登录，先弹登录引导
(3) 滚动时FAB半透明化，点击恢复

### 2-2 游戏圈详情页

入口：社区首页游戏圈列表点击 / 游戏详情页社区tab

#### ① 顶部区域

```
[← 返回]  [🔍]  [⋯]
┌──────────────────────────────┐
│  [游戏封面]  游戏名称          │
│             12.3万关注 | 5.8万帖子 │
│             [+ 关注]          │
└──────────────────────────────┘
综合 | 精选 | 攻略 | 配置 | 存档 | MOD
```

(1) 游戏封面：圆角，48×48
(2) 关注按钮：未关注→"+ 关注"(金色描边)；已关注→"已关注"(灰底)
(3) sub-tab：横向可滚动，选中态加粗+底部指示条

#### ② sub-tab说明

| tab | 内容 | 筛选条件 |
|-----|------|---------|
| 综合 | 该游戏圈全部帖子，按时间倒序 | game_id = 当前游戏 |
| 精选 | 版主/运营标记为精选的帖子 | game_id + is_featured = true |
| 攻略 | 分类标签为"攻略"的帖子 | game_id + tag = 攻略 |
| 配置 | 分类标签为"配置分享"的帖子 | game_id + tag = 配置 |
| 存档 | 分类标签为"存档分享"的帖子 | game_id + tag = 存档 |
| MOD | 分类标签为"MOD"的帖子 | game_id + tag = MOD |

(1) 帖子列表：单列布局，卡片样式
(2) 帖子卡片：头像+昵称+时间 / 标题+正文预览(2行) / 图片(最多3张缩略图) / 点赞+评论+收藏数
(3) 精选帖卡片左上角显示"精选"金色标签
(4) 配置/存档/MOD帖卡片显示对应分类角标

#### ③ 帖子排序

每个sub-tab支持排序切换：

| 排序 | 逻辑 |
|------|------|
| 最新 | 按发布时间倒序（默认） |
| 最热 | 按(点赞数×2 + 评论数×3 + 收藏数×1)加权排序 |

(1) 排序切换在sub-tab下方，小字按钮"最新 | 最热"

### 2-3 游戏详情页-社区tab

入口：游戏详情页顶部tab"社区"

#### ① 竖屏模式

详情页顶部两个tab：详情 | 社区

点击"社区"tab：
(1) 加载该游戏的游戏圈内容（= 游戏圈详情页的简化版）
(2) 显示sub-tab：综合 | 精选 | 攻略 | 配置
(3) 帖子列表同游戏圈详情
(4) 底部"查看全部 →"跳转至完整游戏圈详情页

#### ② 横屏模式

左侧50%：游戏详情
右侧50%：社区内容

(1) 右侧内容 = 游戏圈详情简化版
(2) 支持在右侧直接发帖（点击FAB）

### 2-4 发帖页

入口：FAB按钮 / 游戏圈内发帖

#### ① 页面结构

```
[取消]        发布帖子        [发布]
──────────────────────────────
关联游戏：[选择游戏 ▼]
分类标签：[讨论] [攻略] [配置] [存档] [MOD] [求助]
──────────────────────────────
标题（选填，最多50字）

正文内容...
最多支持2000字

[📷 图片(最多9张)] [#话题] [@用户]
```

#### ② 字段说明

| 字段 | 必填 | 说明 |
|------|------|------|
| 关联游戏 | 是 | 点击后弹出游戏选择列表（搜索+热门），关联后帖子归属该游戏圈 |
| 分类标签 | 是 | 单选，默认"讨论"。选择"存档"时弹出存档上传组件；选择"MOD"时弹出文件上传组件 |
| 标题 | 否 | 最多50字 |
| 正文 | 是 | 最少10字，最多2000字 |
| 图片 | 否 | 最多9张，单张≤10MB，支持JPG/PNG/GIF |

(1) "发布"按钮：正文≥10字且已选择关联游戏时高亮可点击
(2) 发布成功后跳转至帖子详情页
(3) 发布失败弹toast提示原因

#### ③ 存档/MOD附件

若分类标签选择"存档"或"MOD"：

(1) 正文区域下方出现"上传文件"区域
(2) 支持上传zip/rar/7z，单文件≤500MB
(3) 上传后显示文件名+大小+上传进度
(4) 必须填写"适用游戏版本"字段

(3) 点赞/收藏：即时反馈，动画效果
(4) 更多菜单(⋯)：不感兴趣 / 举报 / 复制链接。若为自己的帖子：编辑 / 删除 / 置顶(版主)
(5) 评论区默认显示2条回复，点击"展开更多回复"加载全部（楼中楼）

   - 提交视频链接至平台审核
   - 按播放量阶梯奖励：

| 播放量 | 奖励 |
|--------|------|
| 1,000+ | ¥10 + 100积分 |
| 10,000+ | ¥50 + 500积分 |
| 100,000+ | ¥200 + 2000积分 + 专属徽章 |

(3) 审核周期：提交后3个工作日内
(4) 奖励发放：审核通过后7个工作日内，积分即时到账，现金月结
