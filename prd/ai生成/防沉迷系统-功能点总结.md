# 防沉迷系统功能点总结

## 背景

盖世游戏上架小米渠道，因包含云游戏服务，需接入国家防沉迷实名认证系统。

## 涉及范围

- 仅国内包，仅云游戏相关功能
- 仅C端

---

## 一、实名认证（升级现有软实名为硬实名）

- 接入中宣部「游戏防沉迷实名认证系统」，验证身份证真伪
- 触发时机：用户点击「秒玩」启动云游戏时
- 形式：弹窗（盾牌icon + 法规说明 + 姓名/证件输入 + 提交认证）
- 底部提示港澳台/国外证件用户可联系官网QQ

## 二、启动云游戏的4种场景

| 场景 | 处理 |
|---|---|
| 已实名 + 成年人 | 无感进入，无弹窗 |
| 已实名 + 未成年人 | 判断是否在可玩时段，在则放行，不在则拦截 |
| 已实名 + 信息不匹配 | 弹出实名认证弹窗（同未实名） |
| 未实名 | 弹出实名认证弹窗 |

## 三、未成年人游戏时间限制

- 仅允许在**周五、周六、周日及法定节假日的 20:00-21:00** 进行游戏
- 非可玩时段点击秒玩：直接拦截，弹窗提示
- 游戏中到达21:00：提前15分钟顶部横幅预警，到时强制退出
- 所有拦截弹窗含「退出账号」按钮，方便切换成年人账号

## 四、充值额度限制

触发位置：充值中心页点击「确认支付」时，仅超限时拦截。

| 年龄段 | 单次上限 | 月度累计上限 |
|---|---|---|
| 8周岁以下 | 禁止充值 | 禁止充值 |
| 8-16周岁 | 50元 | 200元 |
| 16-18周岁 | 100元 | 400元 |
| 成年人 | 无限制 | 无限制 |

## 五、个人中心联动

- 头像旁实名标签状态更新：未实名 / 已实名 / 未成年 / 实名异常
- 新增防沉迷状态信息卡片

## 六、运营侧准备

1. 节假日日历维护（影响未成年人可玩日期判定）
2. 隐私政策更新（补充实名信息用途说明）
3. 小米渠道审核材料准备
4. QQ群链接配置（实名弹窗底部跳转）
