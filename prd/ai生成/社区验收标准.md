| # | 验收项 | 通过条件 |

## 4. 异常验收

| # | 验收项 | 通过条件 |
|---|--------|---------|
| 66 | 网络断开-浏览 | 断网时打开社区→显示"网络异常，点击重试"占位图+重试按钮→恢复网络后点击重试→正常加载 |
| 67 | 网络断开-发帖 | 断网时点击发布→toast"网络异常，请检查网络后重试"→帖子内容保留在编辑器中不丢失 |
| 68 | 网络断开-点赞 | 断网时点赞→前端即时变红(乐观更新)→恢复网络后自动同步→若同步失败则回退为未点赞状态 |
| 69 | 接口超时 | 帖子列表加载超过5秒→显示"加载超时，点击重试" |
| 70 | 图片上传失败 | 单张图片上传失败→该图片缩略图上显示红色感叹号+文字"重试"→点击重新上传→不影响其他图片 |
| 71 | 存档文件上传超时 | 上传进度停滞超过30秒→提示"上传超时，建议在WiFi环境下重试"→文件保留可重试 |
| 72 | 帖子已被删除 | 通过链接/推送打开一条已删除的帖子→显示"该内容已被删除"空态页+返回按钮 |
| 73 | 游戏圈不存在 | 通过链接打开一个不存在的游戏圈→显示"该游戏圈不存在"空态页 |
| 74 | 服务端500错误 | 任何接口返回500→对应区域显示"服务异常，请稍后重试"→不影响页面其他已加载区域 |
| 75 | 发帖内容违规被拦截 | 机审命中违规关键词→发布失败→toast"内容包含违规信息，请修改后重试"→编辑器内容保留 |
| 76 | 存档文件安全检测未通过 | 上传完成后安全扫描不通过→通知用户"文件安全检测未通过，无法发布"→帖子不上架 |
| 77 | 断点续传 | 存档文件下载到50%时网络断开→恢复网络→点击继续下载→从50%处继续，不重新开始 |

## 验收checklist使用说明

- 共88条验收项
- 功能验收(45条)：核心逻辑是否正确
- 展示验收(10条)：UI展示是否符合规范
- 交互验收(10条)：操作反馈是否符合预期
- 异常验收(12条)：异常场景是否有兜底
- 埋点验收(11条)：数据上报是否完整

测试走查时逐条标记 ✅/❌，未通过项记录实际表现和截图。
