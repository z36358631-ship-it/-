# 盖世游戏 Mac 端手游库功能点总结

- 盖世游戏 Mac 客户端在“游戏库”下方新增独立侧边栏入口“手游库”。
- Mac 侧边栏顺序为探索、云游戏、数据、找游戏、游戏库、手游库和订单中心。
- 用户进入本模块时只高亮“手游库”，“游戏库”不联动高亮。
- Mac 侧边栏底部保留搜索、下载管理和用户头像。
- “手游库”页面不显示平台一级 Tab。
- “手游库”业务 Tab 为“已安装”和“IPA 资源”。
- “手游库”页面头部只显示“手游库”主标题和业务 Tab，不显示副标题或 App 库路径。
- 原“游戏库”继续承接 PC 游戏，不因新增手游库改变页面和业务规则。
- 切换“游戏库”和“手游库”不终止正在进行的 IPA 解析、下载、校验或安装任务。
- 页面不提供“全部、游戏、工具、娱乐、社交、创意、其他”等应用分类。
- “应用程序类型”只作为单个 App 设置元数据，不作为页面分类筛选。
- 用户只授权一个本地 App 库路径。
- App 库路径只允许选择可读写目录。
- App 库路径失效时保留上次扫描缓存并引导重新授权。
- “已安装”支持按 App 名称或 Bundle ID 搜索。
- “已安装”搜索支持 App 名称或 Bundle ID 模糊匹配。
- “已安装”搜索框位于网格/列表切换右侧和工具栏最右侧。
- “已安装”搜索框缺省提示使用弱提示层级。
- “已安装”工具栏常驻“导入 IPA”按钮。
- “已安装”在空库和非空库状态都显示“导入 IPA”。
- “已安装”初始空状态显示“暂无 App”和“导入 IPA”按钮。
- “已安装”搜索无结果时只提示调整关键词，不显示空状态导入按钮。
- “已安装”支持手动刷新本地 App 库。
- “已安装”刷新只重新扫描目录，不修改 App 数据。
- “已安装”支持网格和列表两种视图。
- “已安装”网格展示图标、名称和关键状态。
- “已安装”网格不显示版本号。
- App 名称在网格中超出一行时截断，悬停可查看完整名称。
- “已安装”不显示 App 总数和右键操作常驻说明。
- “已安装”列表展示图标、名称、Bundle ID、版本、最近使用时间和状态。
- 用户单击 App 可选中当前 App。
- 用户点击空白区域可取消 App 选择。
- 未选中 App 时工具栏“设置”按钮置灰。
- 选中 App 后工具栏“设置”按钮启用。
- App 工具栏“设置”和右键“设置”打开同一个设置窗口。
- App 右键菜单第一项为“设置”。
- App 右键菜单支持“显示应用数据”。
- App 右键菜单支持“在访达中查看”。
- App 右键菜单支持“清除应用数据”。
- App 右键菜单支持“清除应用偏好设置”。
- App 右键菜单支持“清除 PlayChain 数据”。
- App 右键菜单支持“卸载应用”。
- App 右键菜单不提供键盘映射布局的导入与导出。
- “显示应用数据”和“在访达中查看”属于只读导航操作。
- 清除应用数据前必须单独确认。
- 清除应用偏好设置前必须单独确认。
- 清除 PlayChain 数据前必须单独确认。
- 卸载应用前必须单独确认。
- 一次危险操作确认不得授权其他危险操作。
- 卸载时可选择是否同时清理 PlayChain 数据。
- 卸载时可选择是否同时清理 Entitlements。
- 卸载时可选择是否同时清理应用设置。
- 卸载时可选择是否同时清理键盘映射。
- 卸载时可选择是否同时清理应用数据。
- 卸载默认勾选 PlayChain、Entitlements 和应用设置。
- 卸载默认不勾选键盘映射和应用数据。
- App 设置窗口标题展示当前 App 名称和图标。
- App 设置包含“键盘映射”页。
- App 设置包含“图像设置”页。
- App 设置包含“绕过”页。
- App 设置包含“杂项”页。
- App 设置包含“详细信息”页。
- 键盘映射页支持启用或关闭键盘映射。
- 键盘映射页支持智能键盘映射。
- 键盘映射页支持鼠标滚轮映射。
- 键盘映射页支持调整鼠标灵敏度。
- 图像设置页支持选择 iOS 机型。
- 图像设置页支持自动或手动选择分辨率。
- 图像设置页展示当前检测到的分辨率。
- 图像设置页支持调整分辨率缩放。
- 图像设置页支持修复窗口化显示问题。
- 图像设置页支持禁用显示器息屏。
- 图像设置页支持允许旋转。
- 绕过页支持启用 PlayChain。
- 绕过页支持绕过越狱检测。
- 绕过页支持禁用应用内省库。
- 绕过页支持设置 iOS 框架注入模式。
- 杂项页支持 Discord Rich Presence。
- 杂项页支持 Metal HUD。
- 杂项页支持 LLDB 调试。
- 杂项页支持注入 PlayTools。
- 杂项页支持 root 工作目录。
- 杂项页支持设置应用程序类型元数据。
- 详细信息页展示 Bundle 名称。
- 详细信息页展示 Bundle ID。
- 详细信息页展示版本和 Build。
- 详细信息页展示最低 iOS 版本。
- 详细信息页展示 App 本地路径。
- 详细信息页展示显示名称别名。
- App 设置窗口支持重置当前 App 设置。
- App 设置窗口支持单独重置键盘映射。
- App 设置弹窗的关闭按钮固定在右上角。
- App 设置弹窗不显示底部“关闭”按钮。
- App 设置弹窗底部只显示“重置设置”和“重置键盘映射”。
- App 设置弹窗的两个重置按钮统一右对齐。
- 本地 IPA 导入拉起 macOS 原生文件选择器。
- 本地 IPA 文件选择器只允许选择 `.ipa` 文件。
- 本地 IPA 文件选择器支持单个选择。
- 本地 IPA 文件选择器支持批量选择。
- 用户取消本地文件选择后回到原页面且不报错。
- 本地 IPA 解析后进入二级资源选择弹窗。
- 本地 IPA 二级选择默认不勾选任何资源。
- 本地 IPA 二级选择支持逐项勾选。
- 本地 IPA 二级选择支持全选和取消全选。
- 二级选择的“全选/取消全选”位于候选列表容器上方。
- 损坏、重复、不兼容和非 IPA 项在二级选择中置灰并说明原因。
- 未选择任何有效项时“导入”按钮置灰。
- 用户最终确认后才创建本地 IPA 导入任务。
- “IPA 资源”左侧固定展示来源树。
- 来源树根节点为“IPA 资源库”，代表全部启用来源。
- 来源树为每个已添加来源展示独立子节点。
- “IPA 资源库”全库为空时显示“暂无 IPA 资源”和“添加来源”按钮。
- 当前来源为空时显示“当前来源暂无资源”和“添加来源”按钮。
- “IPA 资源”搜索无结果时只提示调整关键词，不显示“添加来源”按钮。
- 同名或同域名来源仍保持独立节点。
- 每个来源使用稳定 `sourceId`，不使用域名或显示名作为唯一键。
- 点击来源树根节点展示全部启用来源的聚合结果。
- 点击单个来源节点只展示该来源资源。
- 根节点按 Bundle ID 聚合去重。
- 来源顺序决定同版本资源的聚合优先级。
- 当前来源失效或被删除时自动返回根节点。
- “IPA 资源”支持按名称或 Bundle ID 搜索。
- “IPA 资源”搜索支持名称或 Bundle ID 模糊匹配。
- “IPA 资源”搜索框位于网格/列表切换右侧和工具栏最右侧。
- “IPA 资源”搜索框缺省提示与“已安装”保持一致。
- “IPA 资源”支持手动刷新当前导入路径。
- 根节点刷新全部启用来源。
- 单个来源节点只刷新当前来源。
- 来源刷新失败时保留上次成功缓存。
- 来源刷新不终止正在进行的下载、校验或安装任务。
- “IPA 资源”支持按字母升序或降序排序。
- “IPA 资源”支持网格和列表两种视图。
- IPA 网格展示图标、名称、版本、状态、唯一主操作和必要进度。
- IPA 列表展示操作、图标、名称、Bundle ID、版本、大小和状态。
- IPA 网格和列表均支持单击选择。
- IPA 网格和列表均支持点击空白取消选择。
- IPA 网格和列表均支持右键。
- IPA 网格和列表均支持双击开始下载或执行唯一主操作。
- 切换列表和网格后保留来源筛选、搜索、排序和任务状态。
- 未选中 IPA 资源时工具栏“应用信息”按钮置灰。
- 选中 IPA 资源后工具栏“应用信息”按钮启用。
- IPA 资源右键菜单只包含“应用信息”。
- IPA 右键“应用信息”和顶部信息按钮打开同一窗口。
- IPA 应用信息弹窗的关闭按钮固定在右上角。
- IPA 应用信息弹窗不显示底部“完成”按钮。
- IPA 应用信息弹窗只通过右上角“X”关闭。
- IPA 应用信息展示 Bundle 名称。
- IPA 应用信息展示 Bundle ID。
- IPA 应用信息展示版本。
- IPA 应用信息展示 iTunes 地址。
- IPA 应用信息展示 IPA 地址。
- IPA 应用信息展示校验和。
- IPA 应用信息展示来源和来源 URL。
- IPA 应用信息展示体积、兼容性和风险。
- App 图标加载失败时显示占位图标。
- 图标加载失败不得把 IPA 标记为无效。
- 用户可以添加无需账号和令牌的公开 HTTPS JSON 来源。
- 来源 URL 变化后立即清除旧验证结果。
- 来源验证反馈区域固定高度，避免弹窗抖动。
- 来源地址、JSON 或目标资源无效时提示“链接无效，JSON 无效或未找到！”。
- 来源验证成功后进入资源二级选择，不直接保存来源。
- 来源资源二级选择默认不勾选任何资源。
- 来源资源二级选择支持逐项选择和全选。
- 用户最终确认后才保存来源并添加选中资源。
- 用户取消来源资源选择时不保存来源。
- 来源管理支持启用和停用来源。
- 来源管理支持删除来源。
- 来源管理支持上移和下移来源。
- 用户选中来源节点后，可通过页面工具栏刷新当前来源。
- 来源管理展示来源 URL 和最近成功缓存时间。
- 来源管理弹窗不显示底部“完成”按钮。
- 来源管理弹窗只通过右上角“X”关闭。
- 确认、提交和导入类弹窗继续保留底部业务操作按钮。
- 停用或删除来源不会卸载已经安装的 App。
- IPA 状态包含未下载。
- IPA 状态包含排队中。
- IPA 状态包含下载中。
- IPA 状态包含已暂停。
- IPA 状态包含校验中。
- IPA 状态包含可安装。
- IPA 状态包含安装中。
- IPA 状态包含已安装同版本。
- IPA 状态包含本地版本更高。
- IPA 状态包含可更新。
- IPA 状态包含下载失败。
- IPA 状态包含校验失败。
- IPA 状态包含安装失败。
- IPA 状态包含不兼容。
- IPA 状态包含缺少来源哈希。
- 未下载资源的唯一主操作为“下载”。
- 排队中资源的唯一主操作为“取消排队”。
- 下载中资源的唯一主操作为“暂停”。
- 已暂停资源的唯一主操作为“继续”。
- 校验中资源不提供可点击主操作。
- 可安装资源的唯一主操作为“安装”。
- 安装中资源不提供可点击主操作。
- 已安装同版本资源不重复安装。
- 本地版本更高时不自动降级。
- 可更新资源的唯一主操作为“更新”。
- 下载失败资源的唯一主操作为“重试”。
- 校验失败资源的唯一主操作为“删除并重下”。
- 安装失败资源的唯一主操作为“重试安装”。
- 不兼容资源的唯一主操作为“查看原因”。
- 缺少来源哈希的资源允许下载但安装前必须确认。
- 每个资源的暂停、继续、取消和重试只影响自己。
- 切换 Tab、来源、搜索、排序和视图不会终止任务。
- 用户取消安装确认后保留已下载包并显示“可安装”。
- 安装失败时保留已校验包并支持重试安装。
- App 更新使用独立下载和安装任务。
- App 更新过程中旧版本保持可启动。
- 新版下载、校验或安装失败时保留旧版本和用户数据。
- 下载完成后校验包内 Bundle ID 与来源元数据一致。
- 同版本资源优先选择有校验和、可访问且来源排序靠前的条目。
- 标注版左侧可快速进入 App、IPA、右键、设置、来源、信息和异常状态。
- 标注版右侧分别展示交互说明和异常边界。
- 标注版支持折叠和恢复右侧说明面板。
- 本地 IPA 在解析候选项后检测包内是否存在 `tersafe.framework`。
- 远程 IPA 在下载完成、安装任务创建前检测包内是否存在 `tersafe.framework`。
- 命中 `tersafe.framework` 时只标记“检测到 ACE 风险特征”，不宣称必然封号或已覆盖所有反作弊。
- 未命中 `tersafe.framework` 不显示“安全”，也不承诺不存在其他账号风险。
- ACE 风险弹窗明确说明可能触发账号风控、处罚或封号风险。
- ACE 风险弹窗明确说明盖世游戏不承担因第三方 IPA 导致的账号封禁损失。
- ACE 风险弹窗提供“取消安装”和“了解风险，仍要安装”两个业务操作。
- 用户取消 ACE 风险弹窗后返回原流程并保留 IPA 勾选或可安装状态。
- 用户的 ACE 风险确认按 IPA SHA-256 记录，同一包不重复提示。
- IPA 哈希变化或新版本上线后必须重新展示 ACE 风险弹窗。
- 已安装但无对应 SHA-256 确认记录的高风险 App 在首次启动前补强提醒。
