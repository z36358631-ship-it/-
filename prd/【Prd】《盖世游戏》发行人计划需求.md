# 二、 背景与目标

**背景：**
当前盖世游戏的核心增长依赖自然流量获客（日均新增8.8万），但留存率极低（D7=3.7%），外部曝光和用户认知几乎为零。传统买量成本高且效果不可控，而竞品233乐园已跑通"发行人计划"模式——通过任务悬赏机制调动UGC创作者在外站（抖音/B站/快手/小红书）发布推广视频，以视频数据表现（点赞量）为结算依据，用平台代币激励创作者。该模式实现了低成本按效果付费的推广闭环。

**目标：**
搭建盖世游戏「发行人计划」功能，实现**"任务发布→创作者领取→外站投稿→数据结算→代币激励"**的完整闭环。支持官方后台发布任务和个人用户APP端发布任务（需审核），通过盖世币经济体系驱动创作者持续为平台游戏产出推广内容。

**核心挑战：** 外站视频数据API抓取的稳定性与准确性、盖世币经济系统的充值/结算/提现闭环、个人发布任务的审核与风控机制、防刷量作弊。

# 三、 故事介绍

## 3.1 用户与运营场景

**用户与运营场景：**

- **C端创作者（小明）：** 小明在盖世游戏APP进入「发行人计划」，看到广场上有一条官方任务"中奖概率倍儿高啊啊"，单价10盖世币，点赞≥1000可获50盖世币。他点击领取任务，去抖音拍了一条15秒游戏视频发布，回到APP粘贴视频链接提交。7个工作日后系统通过API抓取到视频点赞1200，自动结算50盖世币到账。小明累积到1000盖世币后发起提现，1-3个工作日内10元到账支付宝。

- **C端发布者（张三）：** 张三是一个独立游戏开发者，想推广自己的新游戏。他在APP端充值5000盖世币（=50元），创建任务：填写任务名称、搜索选择推广游戏（带图标）、选择投稿平台（抖音+B站）、设定奖金单价10盖世币、系统自动计算单条上限1000盖世币、填入奖池预算5000。提交后24小时内完成审核上架，创作者开始领取他的任务。

- **B端运营（李四）：** 李四在管理后台创建官方推广任务，无需审核直接上架。他可以查看数据看板（领取率、提交率、总播放量、代币消耗），根据效果决定追加预算或提前终止任务。

## 3.2 价值分析

**价值分析**

- **低成本获客引擎：** 将游戏推广从平台自己花钱买量，变为创作者主动拍视频赚收益的众包模式，按效果付费，CPM远低于传统买量。

- **外部曝光与品牌认知：** 创作者在抖音/B站/快手等主流平台发布游戏视频，为盖世游戏带来外部自然流量和用户认知。

- **内容生态正循环：** 游戏获得曝光→创作者获得收入→平台获得新用户→新用户产生更多游戏数据→推动更多推广任务，形成飞轮。

## 3.3 核心体验路径

**创作者&发布者核心链路：**

- **C端创作者核心链路：**
浏览任务广场 -> 查看任务详情（要求/阶梯奖励） -> 领取任务 -> 去外站拍摄发布视频 -> 回到APP粘贴链接提交 -> 平台审核 -> 任务到期后API抓取数据 -> 按阶梯规则结算盖世币 -> 提现或兑换。

- **C端发布者核心链路：**
充值盖世币 -> 创建任务（名称/游戏/平台/单价/上限/预算/有效期） -> 提交审核（24小时内） -> 上架后查看数据 -> 任务到期系统自动结算 -> 剩余预算退回。

- **B端运营核心链路：**
后台创建任务（无需审核直接上架） -> 监控数据看板 -> 审核创作者提交的视频 -> 审核个人发布的任务 -> 追加预算或提前终止。

## 3.4 产品指标预测

**产品指标预测：**

1. **任务广场曝光 -> 任务领取率：** 首月预测为 `15%` 左右（参考233乐园同类数据）。

2. **任务领取 -> 视频提交率：** 首月预测为 `40%`（考虑72小时提交窗口期内的完成率）。

3. **单任务平均带来的外站播放量：** 预测 `5000-20000`（取决于任务奖励力度和创作者粉丝量级）。

4. **创作者次月回访率：** 预测 `30%+`（盈利闭环跑通后，创作者有持续动力）。

## 3.5 路径规划

- **当前版本（V1.0）：** 跑通"官方发任务+个人发任务+创作者领取+视频提交+API数据抓取+盖世币结算+提现"的完整闭环。

- **未来可能升级（预告）：** 创作者等级体系、邀请好友奖励、任务竞标机制、视频内容AI审核、多轮结算（持续追踪视频数据增长）。



# 四、概要设计

## 4.1 模块设计（系统架构图概览）

- **【C端任务体系模块】**

    - **找任务（Tab1）：** 任务卡片列表、官方任务标签与置顶、悬浮球"+ 发布"入口、右上角"玩法说明"。

    - **任务详情：** 产品介绍、投稿平台、投稿要求、奖励阶梯、结算说明、收入排行。

    - **做任务（Tab2）：** 我参与的/我发布的任务管理、编辑/取消/查看操作、"+ 发布任务"入口。

    - **投稿提交：** 选择投稿平台、粘贴视频链接、提交审核。

    - **任务发布：** 创建任务表单（名称/游戏搜索/产品介绍/平台多选/单价/上限/预算/有效期）。

- **【C端经济系统模块】**

    - **钱包（Tab3）：** 可提现金额+提现按钮、盖世币余额、收支明细（收入/消耗/提现/充值）。

    - **充值：** 预设档位+自定义金额、支付方式（微信/支付宝）、充值协议确认（默认不勾选）。

    - **提现：** 最低门槛校验、实名认证、到账方式、税务提示。

- **【B端管理后台】**

    - **数据看板：** 核心指标卡片、任务效果排行、平台投稿分布。

    - **任务管理：** 创建官方任务（免审核）、筛选/搜索、终止任务。

    - **任务审核：** 个人发布任务审核队列（24小时承诺）。

    - **视频审核：** 视频内容审核队列，含外站链接预览。

    - **结算管理：** 结算记录列表、手动执行结算、导出。

    - **风控中心：** 刷量检测、关联账号识别、视频可访问性校验。

    - **创作者审核：** 创作者资质审核、封禁/解封管理。
